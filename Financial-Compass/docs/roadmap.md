# Roadmap

## Фаза 1 — MVP

- [ ] Авторизация и роли пользователей
- [ ] CRUD транзакций (доходы/расходы)
- [ ] Категории и отделы
- [ ] Базовый дашборд с ключевыми метриками
- [ ] Фильтрация по дате, категории, отделу

## Фаза 2 — Бюджетирование

- [ ] Создание и управление бюджетами
- [ ] Контроль план/факт по отделам
- [ ] Уведомления при превышении бюджета
- [ ] Сравнительные графики (план vs факт)

## Фаза 3 — Отчётность

- [ ] Отчёт P&L (прибыли и убытки)
- [ ] Отчёт о движении денежных средств (cash flow)
- [ ] Экспорт отчётов в PDF / Excel
- [ ] Автоматическая рассылка отчётов

## Фаза 4 — Аналитика и прогнозы

- [ ] Анализ трендов расходов/доходов
- [ ] Прогнозирование cash flow
- [ ] KPI-панель для руководства
- [ ] Интеграция с unit-экономикой (Unit Economics Calculator)

## Фаза 5 — Интеграции

- [ ] Импорт банковских выписок
- [ ] Интеграция с 1С / бухгалтерией
- [ ] Интеграция с FinDirector
- [ ] API для внешних систем
