# Financial Compass

Финансовый навигатор для компании **Karina Media**.

## О проекте

Financial Compass — это инструмент финансового планирования и аналитики, разработанный для управления финансовыми потоками Karina Media. Система помогает принимать обоснованные финансовые решения на основе данных.

## Цели проекта

- Централизованный учёт доходов и расходов
- Визуализация финансовых показателей
- Прогнозирование денежных потоков
- Контроль бюджета по отделам и проектам
- Формирование финансовой отчётности

## Основные модули

| Модуль | Описание |
|--------|----------|
| **Dashboard** | Главная панель с ключевыми финансовыми метриками |
| **Бюджетирование** | Планирование и контроль бюджетов |
| **Денежные потоки** | Учёт и прогноз cash flow |
| **Отчёты** | Генерация финансовых отчётов (P&L, баланс) |
| **Аналитика** | Анализ трендов, KPI, unit-экономика |

## Технологический стек

- **Frontend:** React
- **Backend:** Node.js
- **База данных:** PostgreSQL
- **Визуализация:** Chart.js / Recharts
