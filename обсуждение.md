# Обсуждение: Команда Нейро-Агентов KARINA Media

## Цель
Создание полноценной команды AI-агентов, каждый из которых выполняет определённую роль в компании.

---

## Утверждённые роли

| # | Роль | Зона ответственности |
|---|------|----------------------|
| 1 | **CEO** | Стратегия, принятие решений, координация всей команды |
| 2 | **Маркетолог** | Маркетинговая стратегия, контент-план, реклама, аналитика кампаний |
| 3 | **Project Manager** | Управление проектами, задачами, сроками, коммуникация между агентами |
| 4 | **UX-специалист** | Дизайн и создание сайтов, прототипирование, пользовательский опыт |
| 5 | **SEO-специалист** | Поисковая оптимизация, семантическое ядро, анализ конкурентов |
| 6 | **Критик / QA-аналитик** | Анализ и оценка результатов каждого проекта, выявление слабых мест, рекомендации по улучшению |

## Новая роль: Web-Парсер / Аналитик конкурентов

| # | Роль | Зона ответственности |
|---|------|----------------------|
| 14 | **Web-Парсер / Research-агент** | Парсинг интернета, поиск лучших сайтов по нише, анализ дизайна, структуры, UX-паттернов конкурентов, передача данных UX-специалисту для создания сайтов |

**Как работает:**
1. Получает задачу: ниша + регион + ключевые запросы
2. Находит ТОП-20 сайтов в нише (по поиску, рейтингам, трафику)
3. Анализирует: дизайн, структуру, навигацию, цветовые схемы, CTA, карточки товаров, формы
4. Формирует отчёт с рекомендациями для UX-специалиста
5. UX-специалист применяет лучшие практики при создании сайта

## Приоритет №1 — Сайты для бизнесов KARINA Media

### Список сайтов к разработке

| # | Проект | Тип сайта | Ключевые страницы | Статус |
|---|--------|-----------|--------------------|--------|
| 1 | **Интернет-магазин** | E-commerce | Каталог, карточка товара, корзина, оформление заказа, личный кабинет | ⬜ Планирование |
| 2 | **Образование за рубежом** | Информационный + лидогенерация | Страны, программы, университеты, калькулятор стоимости, заявка на консультацию | ⬜ Планирование |
| 3 | **БАДы** | Лендинг + каталог | Каталог продукции, описание составов, сертификаты, отзывы, заказ | ⬜ Планирование |
| 4 | **Салон красоты** | Бизнес-сайт | Услуги, прайс, мастера, онлайн-запись, галерея работ, отзывы | ⬜ Планирование |
| 5 | **Мобилограф** | Портфолио | Портфолио работ, услуги, тарифы, бронирование съёмки, блог | ⬜ Планирование |
| 6 | **Корейская косметика** | Интернет-магазин | Каталог по брендам, карточки товаров, обзоры, бьюти-блог, доставка | 🟡 В работе |

### Порядок работы по каждому сайту

```
1. Web-Парсер → исследование ниши, анализ ТОП конкурентов
2. SEO-специалист → семантическое ядро, структура сайта
3. Копирайтер → тексты для всех страниц
4. UX-специалист → дизайн и разработка на основе данных парсера
5. Маркетолог → стратегия продвижения
6. Project Manager → контроль сроков и качества
```
